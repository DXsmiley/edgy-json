Edgy-JSON

JSON verification for hipsters

Documentation View on GitHub Download .zip Download .tar.gz

Edgy-JSON is a python alternative to JSON Schema. If you want to introduce some potentially broken code into your project, you should use Edgy-JSON. If not, don't worry. Edgy-JSON will probably become mainstream one day, and I'll have moved on to some other bleeding-edge project.

Install

pip install edgy-json

Use

import edgy
edgy.check(schema, data)
